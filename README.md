### 🍏Sesac coding Test


   - **메모리**
        - 값이 저장되어 있는 메모리의 주소값ㅇ르 이름으로 연결해 놓은 것
        - 데이터를 저장하는 공간</br>
        
   - **변수와 상수**
     - 변수: 값을 변경 가능
     - 상수: 한 번 저장된 값을 다른 값으로 변경할 수 없음

  - **타입형**
       - Int: 정수값을 저장하는 자료형
       - String: 문자열을 저장하는 자료형
       - Bool: true or false를 저장하는 자료형
            - 명백하게 나눌 수 있는 (Ex: pass or nonpass)
       - Double: 실수값(소수점)을 저장하는 자료형
